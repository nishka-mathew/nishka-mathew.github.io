Machine Learning
AI Microalgae Biomass Predictor
Built a predictive ML model to forecast microalgae biomass growth for Ecoclosure's Biochromic Window systems, a living building technology that uses microalgae for energy regulation and carbon capture. Tested linear, logarithmic, and random forest regression across multiple preprocessing strategies first using Kaggle data, then using real sensor data from Charlotte's Innovation Barn, with min-max scaled linear regression achieving the best performance at R² = 0.911. Designed interactive visualizations including a 14-day biomass forecast and feature importance charts to make model outputs interpretable to non-technical stakeholders.

UX Research
MvR — Movie & TV Review App
Conducted a multi-phase UX research study on movie and TV review platforms, identifying key pain points users experience on existing platforms like IMDb and Letterboxd. Starting with structured usability tests and expanding to affinity diagramming, empathy maps, and user personas. The research surfaced five core themes around trust, interface usability, rating systems, content discovery, and personalization. Built on those findings through co-design sessions with 10 participants aged 18–24, where participants sketched and thought aloud through key screens. Synthesized outputs into three design concepts: Trusted Transparency, Streamlined Navigation, and Personalized Discovery. Translated those concepts into higher-fidelity wireframes and prototypes in Figma, producing user journey maps, user flow diagrams, storyboards, and a full project brief communicating the recommended user experience to stakeholders.

CinemaQuest
iOS app built in Xcode using Swift for movie enthusiasts who want to discover new films and keep track of what they have watched. The app pulls real-time trending movie data from The Movie Database (TMDB) API, displaying detailed information including overviews, release dates, ratings, and movie posters. Users can manage two personal watchlists (Up Next and Watched) with the ability to add custom movies, leave notes, and set watch dates. Trending movies can also be sorted by release date, popularity, or rating. Built with a UIKit interface featuring async image loading via Nuke, all watchlist data is stored locally for persistent access across sessions.
